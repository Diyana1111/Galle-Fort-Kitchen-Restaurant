# 🏛️ Galle Fort Kitchen

### Responsive Restaurant Website | DecodeLabs Full Stack Internship — Project 1

---

## 📌 Project Overview

**Galle Fort Kitchen** is a responsive frontend website for an authentic Sri Lankan restaurant located in the historic Galle Fort, Galle, Sri Lanka. This project was built as **Project 1** of the DecodeLabs Full Stack Development Internship (Batch 2026).

The goal was to design and build a clean, user-friendly, fully responsive restaurant website using only **HTML, CSS, and JavaScript** — no frameworks, no libraries.

## 🎯 Project Goal

Build a responsive frontend interface for a simple web application that works seamlessly across all screen sizes — mobile, tablet and desktop.

---

## ✨ Features

- Fully responsive layout — mobile, tablet and desktop
- Sticky navigation bar with smooth scroll
- Hamburger menu for mobile view
- Hero section with background image and overlay
- Most Popular dishes section with food cards
- Full menu organized by category — Starters, Mains, Desserts, Drinks
- About section with restaurant story
- Footer with location, contact, and opening hours
- Active nav link highlight on scroll
- Navbar shadow effect on scroll

---

## 📐 Responsive Breakpoints

| Breakpoint | Screen Size | Layout                          |
| ---------- | ----------- | ------------------------------- |
| Mobile     | < 768px     | Single column, hamburger menu   |
| Tablet     | ≥ 768px     | 2 column grid, full nav visible |
| Desktop    | ≥ 1024px    | 3 column grid, full layout      |
